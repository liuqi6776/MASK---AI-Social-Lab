# MASK 产品评审记录

> 日期：2026-07-03
> 评审方：Sisyphus（作者自审）+ GPT-5.5 + Gemini 3.1 Pro 三方独立评审
> 结论：**重构为图灵竞技场**

## 一句话结论

**原设计不是一个产品，而是四个半成品被一个"你分得清是人还是 AI"的噱头强行拼在一起。**

这个噱头同时引爆三颗炸弹：
1. 🔴 **信任崩塌**：AI 潜入情绪社区
2. 🔴 **中国合规红线**：AI 冒充真人不标注 = 违规
3. 🔴 **亏损单元经济**：每帖 AI 图 + 每消息 LLM = 财务自杀

**但它有救**：把核心机制从"社交平台默认态"重构为"用户知情同意的游戏"。

## 三方裁定

| 评审方 | 裁定 | 核心理由 |
|--------|------|---------|
| Sisyphus | BUILD-NARROWER | 砍成图灵游戏可救，能绕开三雷 |
| GPT-5.5 | BUILD-NARROWER | 唯一内聚的产品是图灵游戏，其余全砍 |
| Gemini 3.1 | DON'T-BUILD | 原设计在中国合规上是死路 |

**共识**：作为"AI 混入情绪社交平台"→ 死。作为"用户同意的图灵游戏"→ 可行。

## 砍 / 留 / 重构

| 决策 | 项目 | 原因 |
|------|------|------|
| ✂️ 砍掉 | 心情广场 | 审核噩梦 + 无护城河 |

| ✂️ 砍掉 | 每帖必生成 AI 图 | 成本失控 |
| ✂️ 砍掉 | 向量记忆(Pinecone) | 贵 + 隐私风险，MVP 无状态 |

| ✅ 保留 | 图灵游戏 | 上升为唯一核心 |

| 🔄 重构 | 变现 | 从订阅聊天 → 对局/槽位/赛季 |

## 关键原则

> **"欺骗作为游戏机制可以接受；作为社交平台的默认运行模式，会摧毁信任。"**

- ✅ 游戏："你进入图灵测试，部分对手是 AI"（合法）
- ❌ 社交平台："安慰你的人可能偷偷是 AI"（危险+违规）
